# v2.0 Milestone Audit

## Milestone

`v2.0 Observation Graph Architecture`

Completed: `2026-03-31`

## Outcome

v2.0 shipped the observation-graph architecture:

- quote-first v2 extraction packets
- canonical v2 observations with span support
- structural, coverage, and semantic v2 validators
- deterministic derivation into analyst rows
- additive DuckDB `v2_*` tables
- triple v2 export surface
- 9-deal migration through the v2 pipeline

## Follow-On Work Identified

v2.0 left one repo-hygiene gap that was intentionally handled in the next
milestone instead of stretching the architecture milestone:

- cut over the live repo surface so v2, not v1, is the default
- archive current v1 outputs away from the live `data/skill/<slug>/` surface
- preserve legacy workflows under explicit legacy skill names

That follow-on work shipped as `v2.1 V2 Default Cutover + Legacy Archive`.
