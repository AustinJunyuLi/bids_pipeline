From skill_pipeline/models.py (quote_first raw artifacts):
```python
class RawSkillActorsArtifact(SkillModel):
    quotes: list[QuoteEntry]
    actors: list[RawSkillActorRecord]
    count_assertions: list[RawCountAssertion] = Field(default_factory=list)
    unresolved_mentions: list[str] = Field(default_factory=list)

class RawSkillEventsArtifact(SkillModel):
    quotes: list[QuoteEntry]
    events: list[RawSkillEventRecord]
    exclusions: list[SkillExclusionRecord] = Field(default_factory=list)
    coverage_notes: list[str] = Field(default_factory=list)

class QuoteEntry(SkillModel):
    quote_id: str
    block_id: str
    text: str
```

The key dispatch rule:
- mode == "quote_first" -> use raw_actors.actors, raw_events.events (raw_actors/raw_events are populated; actors/events/spans are None)
- mode == "canonical" -> use actors.actors, events.events (actors/events/spans are populated; raw_actors/raw_events are None)
- mode "legacy" no longer exists

Format detection (extract_artifacts.py lines 50-66):
- If `spans.json` exists on disk -> canonical mode (validates SkillActorsArtifact, SkillEventsArtifact, SpanRegistryArtifact)
- Else if `"quotes" in actors_payload or "quotes" in events_payload` -> quote_first mode (validates RawSkillActorsArtifact, RawSkillEventsArtifact)
- Else -> ValueError

CRITICAL: When quote_first is detected, BOTH actors_payload AND events_payload are validated through their respective Raw models. Both must contain the required `quotes` field even if it is an empty list.

Add the `_cue_is_covered_quote_first` function. Quote-first artifacts have `raw_actors` and `raw_events` with `quote_ids` referencing quotes that carry `block_id`. Coverage checks overlap by comparing cue block_ids against quote block_ids:

```python
def _cue_is_covered_quote_first(cue: CoverageCue, artifacts: LoadedExtractArtifacts) -> bool:
    cue_block_ids = set(cue.block_ids)
    quote_index: dict[str, str] = {}
    for q in artifacts.raw_actors.quotes:
        quote_index[q.quote_id] = q.block_id
    for q in artifacts.raw_events.quotes:
        quote_index[q.quote_id] = q.block_id

    if cue.cue_family == "advisor":
        for actor in artifacts.raw_actors.actors:
            if actor.role != "advisor":
                continue
            if _quote_ids_overlap(cue_block_ids, actor.quote_ids, quote_index):
                return True

    for event in artifacts.raw_events.events:
        if cue.suggested_event_types and event.event_type not in cue.suggested_event_types:
            continue
        if _quote_ids_overlap(cue_block_ids, event.quote_ids, quote_index):
            return True
    return False
```

Add the helper:
```python
def _quote_ids_overlap(
    cue_block_ids: set[str],
    quote_ids: list[str],
    quote_index: dict[str, str],
) -> bool:
    for qid in quote_ids:
        block_id = quote_index.get(qid)
        if block_id and block_id in cue_block_ids:
            return True
    return False
```
