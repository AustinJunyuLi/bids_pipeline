```python
def _dedup_actors(
    actors_dict: dict, events: list[dict]
) -> tuple[dict, list[dict], dict[str, str]]:
    """Deduplicate actors by (canonical_name_normalized, role, bidder_kind).

    Returns (updated_actors_dict, updated_events, actor_dedup_log).
    actor_dedup_log maps removed_actor_id -> survivor_actor_id.
    """
```

Implementation rules:
- Dedup key: `(actor["canonical_name"].strip().upper(), actor["role"], actor.get("bidder_kind"))`
- When a group has >1 actor, pick survivor = actor with the most `evidence_span_ids` (ties broken by first occurrence).
- Merge all `evidence_span_ids` from duplicates into survivor (union, preserving survivor's order first, then appending new ones).
- Merge `aliases` from duplicates into survivor (union).
- Merge `notes` from duplicates into survivor (union).
- Build a remap dict: `{removed_actor_id: survivor_actor_id}`.
- Rewrite `actor_ids` in all events: replace any removed_actor_id with survivor_actor_id.
- Rewrite `advised_actor_id` in all actors: replace any removed_actor_id with survivor_actor_id.
- Rewrite `executed_with_actor_id` in all events: replace any removed_actor_id with survivor_actor_id.
- Rewrite `invited_actor_ids` in all events: replace any removed_actor_id with survivor_actor_id.
- Return: (updated actors_dict with only survivors, updated events, actor_dedup_log).

Insert after `_check_empty_anchor_text` and before `run_check`:

```python
def _check_actor_audit(artifacts: LoadedExtractArtifacts) -> list[CheckFinding]:
    """Audit actor roster for residual issues from chunked extraction."""
    findings: list[CheckFinding] = []
    actors = artifacts.raw_actors.actors if artifacts.mode == "legacy" else artifacts.actors.actors
    events = artifacts.raw_events.events if artifacts.mode == "legacy" else artifacts.events.events
```

Four checks inside _check_actor_audit:

1. **Duplicate canonical_name** -- Group actors by `canonical_name.strip().upper()`. If any group has >1 actor_id, emit a warning finding:
   - check_id: `"duplicate_canonical_name"`
   - severity: `"warning"`
   - description: `f"Actors {[a.actor_id for a in group]} share canonical_name '{name}'."`
   - actor_ids: list of all actor_ids in the group

2. **Bidder with no NDA** -- For each actor with role=="bidder" and is_grouped==False, check if any event with event_type=="nda" has that actor's actor_id in its actor_ids. If not:
   - check_id: `"bidder_no_nda"`
   - severity: `"warning"`
   - description: `f"Bidder '{actor.actor_id}' has no NDA event."`
   - actor_ids: [actor.actor_id]

3. **Count assertion gap** -- Access count_assertions from actors artifact (artifacts.raw_actors.count_assertions if legacy, artifacts.actors.count_assertions if canonical). For each assertion, count matching events or actors. For subject containing "nda" or "confidentiality", count events with event_type=="nda". Compare against assertion.count. If mismatch:
   - check_id: `"count_assertion_gap"`
   - severity: `"warning"`
   - description: `f"Count assertion '{assertion.subject}' expects {assertion.count}, found {actual}."`
   - actor_ids: []

4. **Advisor missing advised_actor_id** -- For each actor with role=="advisor" and advised_actor_id is None:
   - check_id: `"advisor_missing_advised"`
   - severity: `"warning"`
   - description: `f"Advisor '{actor.actor_id}' has no advised_actor_id."`
   - actor_ids: [actor.actor_id]
