## Integration Issues

### CRITICAL: deal-agent SKILL.md missing `gates` step

The deal-agent SKILL.md procedure does not include `skill-pipeline gates --deal <slug>` between coverage and verify-extraction. An agent following the procedure verbatim will skip gates, then enrich-core will fail with `FileNotFoundError: Missing required input: ...gates_report.json`.

**Affected requirements:** GATE-01, GATE-02, GATE-03, GATE-04, DB-02
**Fix:** Add gates step to `.claude/skills/deal-agent/SKILL.md` procedure and Individual Invocation section. Sync mirrors.

### WARNING: deal-agent SKILL.md missing `compose-prompts` step

The procedure does not explicitly call `skill-pipeline compose-prompts` before extract-deal. The extract-deal SKILL.md references it as a preflight, but the deal-agent orchestration is incomplete as a standalone guide.

**Affected requirements:** PROMPT-01 through PROMPT-06, INFRA-05
**Fix:** Add compose-prompts step before extract-deal, or add a cross-reference note.
